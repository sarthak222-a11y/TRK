# NEET Smart Study Planner

A lightweight web-based NEET study planner with:
- Study-wise timer
- Daily task calendar
- Weekly progress tracking
- Syllabus tracker (Theory, PYQ, Revision, Mastered)
- Countdown to NEET
- Test marks analysis with bar graphs

## How to Run
1. Download or clone the repo
2. Open `index.html` in your browser
3. (Optional) Deploy using GitHub Pages

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
